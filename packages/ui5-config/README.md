# @sap-ux/ui5-config

Utilities functions to generate middleware sections in ui*.yaml files



## Installation
Npm
`npm install --save @sap-ux/ui5-config`

Yarn
`yarn add @sap-ux/ui5-config`

Pnpm
`pnpm add @sap-ux/ui5-config`
