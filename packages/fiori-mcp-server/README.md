# @sap-ux/fiori-mcp-server

Library that provides MCP server for fiori tools.


## Installation
Npm   
`npm install --save @sap-ux/fiori-mcp-server`

Yarn   
`yarn add @sap-ux/fiori-mcp-server`

Pnpm   
`pnpm add @sap-ux/fiori-mcp-server`
