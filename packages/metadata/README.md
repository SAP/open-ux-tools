# Metadata

Package containing metadata service

## Installation
Npm
`npm install --save @sap-ux/metadata`

Yarn
`yarn add @sap-ux/metadata`

Pnpm
`pnpm add @sap-ux/metadata`

## Usage

1. Import the needed functions in your modules

    ```typescript
    import { * } from '@sap/ux-metadata';
    ```
