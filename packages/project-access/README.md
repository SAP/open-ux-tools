# @sap-ux/project-access

Library that offers functions to find, read, and modify projects.


## Installation
Npm   
`npm install --save @sap-ux/project-access`

Yarn   
`yarn add @sap-ux/project-access`

Pnpm   
`pnpm add @sap-ux/project-access`
