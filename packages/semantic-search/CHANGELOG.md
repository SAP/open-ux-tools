# Change Log

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/)
and this project adheres to [Semantic Versioning](http://semver.org/).

## [0.1.0] - 2026-01-26

### Added

- Initial release of @sap-ux/semantic-search
- TypeScript implementation with full type support
- Dual CommonJS and ESM module outputs
- Semantic search functionality with embeddings
- Store and load embeddings to/from disk
- Register and manage multiple embedding sets
- Search across single or multiple datasets
- Caching support for improved performance
- ONNX runtime-based embedding generation
- Comprehensive test suite using Jest
